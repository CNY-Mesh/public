About Meshtastic

Meshtastic is an open-source, decentralized mesh network that uses affordable LoRa radios to pass short messages and location data directly between devices—no cell towers, Wi-Fi, or internet required. Every device is a node, and each node relays messages for the others.

How it works

  • LoRa radios: long-range, low-power data links
  • Mesh hops: messages route across multiple nodes
  • Phone app: configure, message, and share GPS via Bluetooth
  • Independent: community-owned, no central gatekeepers

Why it matters

  • Emergency readiness: stay connected during outages
  • Outdoors: hiking, camping, hunting, boating groups
  • Community projects: local, resilient communications
  • Ham experimentation: digital/radio tinkering welcomed

Central New York Opportunity

We’re focusing on Onondaga, Oswego, Madison, Oneida, Cayuga, and Cortland counties, with an eye toward linking with Rochester, Buffalo, Ithaca, and Albany for a true regional mesh backbone.
